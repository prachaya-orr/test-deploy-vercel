ภาค 8: Crypto และ DeFi Manipulation — เกมเดิมในสนามใหม่

บทนำ: ทำไมต้องมีภาคนี้

ในตอนที่หนังสือเล่มนี้ถูกเขียน ตลาด Crypto มีมูลค่ารวมเกินกว่า 3.5 ล้านล้านดอลลาร์ มี Token ใหม่ๆ ถูกสร้างขึ้นวันละหลายร้อยตัว และมีนักลงทุนคนไทยจำนวนมหาศาลเข้ามาในตลาดนี้ — ส่วนใหญ่ "ขาดทุน"

สิ่งที่น่าตกใจคือ เกมการปั่นใน Crypto นั้นเหมือนกับการปั่นหุ้นแทบทุกประการ — เพียงแต่ใช้เครื่องมือใหม่ คำศัพท์ใหม่ และเร็วกว่าหลายเท่า เหยื่อรายย่อยที่อ่านหนังสือเล่มนี้จบ จะพบว่ารูปแบบในตลาด Crypto คือสิ่งที่ตัวเองเคยเรียนรู้ไปแล้วในภาค 1-2

แต่มี "ความแตกต่าง" 5 ข้อที่ต้องเข้าใจก่อนเข้าตลาด:

  1. ไม่มี ก.ล.ต. คุ้มครอง — ตลาด Crypto ส่วนใหญ่ไม่มีหน่วยงานกำกับ การปั่นเป็นเรื่อง "ปกติ" ที่ทำกันได้อย่างเปิดเผย
  2. เปิด 24/7 — ราคาเคลื่อนไหวได้ตลอดเวลา การพักผ่อนคือความเสี่ยง
  3. ผันผวนสูงกว่าหุ้น 5-10 เท่า — ราคาขึ้นลง 20-30% ในวันเดียวเป็นเรื่องปกติ
  4. มี Smart Contract และ Oracle ที่ถูก Hack ได้ — ความเสี่ยงทางเทคนิคที่หุ้นไม่มี
  5. ผู้ปั่นและเหยื่อสามารถเป็นใครก็ได้บนโลก — ไม่จำเป็นต้องอยู่ในประเทศเดียวกัน

คำเตือนสำคัญ: หากคุณยังไม่อ่านภาค 1-3 ของหนังสือเล่มนี้ — โปรดกลับไปอ่านก่อน เพราะภาคนี้สมมติว่าคุณเข้าใจกลไกพื้นฐานของการปั่นหุ้นแล้ว

CEX vs DEX — ใครคุมอะไร

31.1 Rug Pull — การกระชากพรม (รูปแบบที่พบบ่อยที่สุด)

นิยาม: ทีมพัฒนาสร้าง Token หรือโปรเจกต์ขึ้นมา ระดมเงินจากนักลงทุน แล้ว "หายตัว" พร้อมเงินทั้งหมด

กระบวนการ Rug Pull แบบคลาสสิก:

Step 1: สร้าง Token + Smart Contract
        └── Token Supply: 1,000,000,000
        └── ทีมเก็บ 50% (Locked เพื่อหลอก) หรือ Unlocked

Step 2: สร้าง Liquidity Pool บน DEX
        └── ใส่ Token + ETH/BNB ในอัตราส่วน 50:50
        └── ตั้งราคาเริ่มต้นที่ $0.0001

Step 3: การตลาด
        └── จ้าง Influencer พูดถึง
        └── สร้าง Telegram/Discord ที่มี "ความคึกคัก" (จ้างคน)
        └── ปล่อยข่าว "Listing บน Exchange ใหญ่" (ปลอม)
        └── สร้าง Roadmap หรู Whitepaper สวย

Step 4: ดันราคา
        └── ทีมงาน + Whale ซื้อ Token เอง สร้างวอลุ่ม
        └── ราคาพุ่ง 100x-1000x ในไม่กี่วัน
        └── FOMO เริ่มขึ้น คนตามแห่ซื้อ

Step 5: Rug Pull
        └── ทีมงานเทขาย Token ทั้งหมดในเวลาไม่กี่นาที
        └── ราคาดิ่งจาก $1 เหลือ $0.0001
        └── Liquidity Pool ถูกถอนออก
        └── เว็บไซต์หาย Telegram ถูกลบ

สถิติที่น่าตกใจ: จากรายงานของ Chainalysis ปี 2023 — Rug Pull คิดเป็น 37% ของ Crypto Scam ทั้งหมด มูลค่าความเสียหายรวมกว่า $4.2 พันล้าน

Honeypot — กับดักที่ขายไม่ได้

กลไก: Smart Contract ถูกเขียนให้ ใครก็ตามที่ไม่ใช่เจ้าของไม่สามารถขาย Token ได้ เหยื่อซื้อเข้าไปแต่ขายออกไม่ได้

Honeypot Logic ใน Smart Contract:

function transfer(address to, uint amount) {
    if (msg.sender != owner) {
        // ไม่ทำอะไร หรือ revert
        revert("Sale not allowed");
    }
    _transfer(msg.sender, to, amount);
}

วิธีตรวจสอบ:
- ใช้ honeypot.is (Free Tool)
- ดู Transaction ใน Block Explorer — ถ้ามีแต่ "Buy" แต่ไม่มี "Sell" → Honeypot
- ทดลองซื้อด้วยเงินน้อยๆ แล้วลองขายดู (เสี่ยงเสียค่า Gas)

ข้อสังเกต: Honeypot มักวิ่งราคาขึ้นเรื่อยๆ ไม่มีลง เพราะไม่มีใครขายได้ ทำให้ดู "น่าซื้อ" — แต่นั่นแหละคือกับดัก

31.2 Pump and Dump Group — กลุ่มปั่นสาธารณะ

ลักษณะ: กลุ่ม Telegram/Discord/X ที่นัดเวลากันซื้อ Token หนึ่งพร้อมกัน ดันราคาขึ้น แล้วขายทำกำไร

กระบวนการ:

Step 1: Admin ของกลุ่มเลือก Token (มักเป็น Low Cap < $10M)
Step 2: Admin ซื้อ Token ก่อนล่วงหน้า 1-3 วัน (Pre-buy)
Step 3: ประกาศวันเวลา "Pump" ในกลุ่มสมาชิก VIP ก่อน 30 นาที
Step 4: ประกาศในกลุ่มสาธารณะตรงเวลานัด
Step 5: ทุกคนแห่ซื้อพร้อมกัน ราคาพุ่ง 50-200%
Step 6: Admin และ VIP เทขาย ทำกำไร
Step 7: คนที่เข้าช้า "ติดดอย" ราคาดิ่งกลับ

ผลของเกม:
- Admin: กำไร 50-100% (ซื้อก่อน ขายก่อน)
- VIP Members: กำไร 30-50% (รู้ก่อน 30 นาที)
- Public Members: กำไร 0-20% หรือ ขาดทุน 20-50% (เข้าช้า ออกช้า)
- คนที่เข้าหลังประกาศ 5 นาที: ขาดทุน 30-70%

ข้อสังเกต: กลุ่ม Pump and Dump เกือบทั้งหมดออกแบบมาให้ Admin ได้กำไร — สมาชิกเป็นเพียง "เครื่องมือ" สร้างวอลุ่มและดันราคา ถ้าคุณคิดว่าตัวเอง "เร็วพอ" ที่จะกำไร — คุณคือเหยื่อ

Cross-Exchange Pump

รูปแบบขั้นสูง: ปั่นบน Exchange เล็กก่อน รอราคาวิ่ง แล้วค่อย List บน Exchange ใหญ่ แล้วเทขาย

Phase 1 (Exchange เล็ก - Gate.io, MEXC):
  - Token เปิด Trading ที่ราคา $0.01
  - ปั่นราคาขึ้น $1 (100x) ในช่วง 2 สัปดาห์
  - สร้าง "ตำนาน" ว่าเป็น Token ที่ "ทำกำไร 100x"

Phase 2 (Exchange กลาง - KuCoin, Bitget):
  - List บน Exchange กลาง คนที่เห็น "ตำนาน" แห่ซื้อ
  - ราคาดันขึ้นต่ออีก 2-3x

Phase 3 (Exchange ใหญ่ - Binance):
  - List บน Binance — ราคาวิ่งทะลุ Top อีกครั้ง
  - ผู้ปั่นเทขายทั้งหมดในวันแรกของการ List
  - ราคาดิ่ง 70-90% ภายใน 1 สัปดาห์

31.3 Wash Trading บน CEX และ DEX

บน CEX: Exchange เองหรือผู้ปั่นใช้บัญชีหลายบัญชีซื้อขายกันเอง สร้างวอลุ่มปลอม

สถิติช็อค: จากการศึกษาของ Bitwise (2019) — ของวอลุ่ม Bitcoin จาก Exchange ทั้งหมด 95% เป็น Wash Trading มีเพียง 10 Exchange เท่านั้นที่มีวอลุ่มจริง

วิธีตรวจสอบ:
- ใช้ CoinMarketCap, CoinGecko ที่มีตัวกรอง "Adjusted Volume"
- ดู Order Book — ถ้าวอลุ่มสูงแต่ Order Book บาง = ผิดปกติ
- ดู Trade History — ถ้าขนาด order ซ้ำๆ ในเวลาที่ใกล้กันมาก = Bot Wash Trading

บน DEX: สร้าง Wallet หลายตัว ซื้อขายกันเองใน Pool

Wallet A → ขาย 1,000 Token ไปที่ Pool → ราคาลง 5%
Wallet B → ซื้อ 1,000 Token จาก Pool → ราคาขึ้น 5%
(ทั้ง A และ B เป็นเจ้าของเดียวกัน)

ผลลัพธ์: Volume เพิ่ม แต่ราคาไม่เปลี่ยน
ค่าใช้จ่าย: Gas Fee + Slippage (น้อยมาก)

ข้อสังเกต: ใน Crypto บางโปรเจกต์มี "Volume Incentive" — แจกรางวัลให้ Token ที่มีวอลุ่มสูง ทำให้ Wash Trading กลายเป็นเรื่อง "คุ้มค่า" สำหรับผู้ปั่น

31.4 Sandwich Attack — กับดักของ DEX

กลไก: Bot จับ Transaction ที่จะส่งเข้า Mempool แล้ว "ใส่" Transaction ของตัวเองก่อนและหลัง เพื่อกินส่วนต่าง

+------------------------------------------+
|         SANDWICH ATTACK FLOW               |
+------------------------------------------+
|                                            |
|  เหยื่อ: ส่ง Tx ซื้อ 100 ETH                |
|         (ราคาคาดหวัง $3,000)               |
|                                            |
|  Bot เห็น Tx ใน Mempool:                   |
|  └── ใส่ Tx ของตัวเอง (Front-run)         |
|      ซื้อ 50 ETH ก่อน → ราคาขึ้นเป็น $3,015|
|                                            |
|  เหยื่อ Tx ทำงาน:                          |
|  └── ซื้อ 100 ETH ที่ $3,015 (แพงขึ้น)     |
|      ราคาขึ้นเป็น $3,030                   |
|                                            |
|  Bot ขายทันที (Back-run):                  |
|  └── ขาย 50 ETH ที่ $3,030                |
|      กำไร: $0.75 ต่อ ETH × 50 = $37.5     |
|                                            |
|  ผลของเหยื่อ:                              |
|  ซื้อ ETH แพงกว่าที่ควร $1.5 × 100 = $150 |
+------------------------------------------+

วิธีป้องกัน:
1. ตั้ง Slippage ให้ต่ำ (0.5-1%) — ถ้า Slippage สูง Tx จะ Revert
2. ใช้ Private Mempool (Flashbots, MEV Blocker)
3. ใช้ DEX ที่มี MEV Protection (CowSwap, 1inch Fusion)
4. แบ่ง Order ขนาดใหญ่เป็นหลายๆ Order เล็ก
5. หลีกเลี่ยงการเทรด Token ที่มี Liquidity ต่ำ

31.5 Insider Trading ก่อน CEX Listing

กลไก: พนักงาน Exchange หรือคนที่รู้ก่อนว่า Token จะถูก List ไปซื้อก่อนล่วงหน้า

ตัวอย่างที่เคยเกิดขึ้นจริง:
- 2022: อดีตผู้จัดการ Coinbase ถูกจับข้อหา Insider Trading ใช้ข้อมูลภายในซื้อ Token ก่อน Listing 25 ครั้ง กำไรกว่า $1.5 ล้าน
- มีงานวิจัยจาก University of Technology Sydney พบว่า ราคา Token มักวิ่งขึ้น 5-15% ในชั่วโมงก่อน Coinbase ประกาศ Listing

วิธีสังเกต:
- ดู On-chain Activity — ถ้ามี Wallet ขนาดใหญ่ที่ไม่เคยเทรด Token นี้ มาซื้อก่อนข่าว → น่าสงสัย
- ใช้ Nansen Smart Money Tracker
- ดู Volume ผิดปกติใน 24 ชั่วโมงก่อนข่าว

31.6 Liquidity Drain Attack

กลไก: ผู้ปั่นเป็นเจ้าของ Liquidity Pool ส่วนใหญ่ ทำให้สามารถถอน Liquidity ออกได้ทุกเมื่อ ทำให้คนอื่นขาย Token ไม่ได้

ปกติ Pool มี:
├── 10,000 TOKEN
└── 100 ETH ($300,000)

ผู้ปั่นถอน Liquidity 90%:
├── เหลือ 1,000 TOKEN
└── เหลือ 10 ETH ($30,000)

ผลลัพธ์:
├── ราคา TOKEN จาก $30 เหลือ $30 (ดูเหมือนเดิม)
├── แต่ใครจะขาย 100 TOKEN จะ Slippage 100%
├── Token แทบขายไม่ได้
└── นี่คือ "Soft Rug Pull"

วิธีตรวจสอบ:
- ดู Liquidity ของ Pool ที่ DexTools, DexScreener
- ดูว่า LP Token (ใบรับรอง Liquidity) ถูก Lock หรือไม่
- ดูประวัติ Liquidity — ถ้ามีการเพิ่ม-ลด ผิดปกติ → ระวัง

31.7 Memecoin Manipulation

ลักษณะของ Memecoin:
- ไม่มี Utility (ใช้งานจริงไม่ได้)
- ขายความ "ตลก" หรือ "ติดเทรนด์"
- พึ่งกระแสและ Influencer 100%
- ราคาวิ่งจาก Sentiment ล้วนๆ

Lifecycle ของ Memecoin:

Day 0:    Launch — ราคา $0.000001
Day 1-3:  Pump Phase — Influencer พูดถึง ราคาวิ่ง 100-1000x
Day 4-7:  FOMO Phase — รายย่อยแห่ซื้อ ราคาพุ่งสุด
Day 8-14: Distribution Phase — Whale และ Influencer เริ่มเทขาย
Day 15+:  Death Phase — ราคาดิ่ง 90-99% ไม่กลับมาอีก

สถิติของ Memecoin:
- 90% ของ Memecoin หายไปในเวลา 6 เดือน
- จากการวิเคราะห์ของ Chainalysis: คนที่ถือ Memecoin > 1 เดือน — 95% ขาดทุน
- เฉพาะ Top 1% ของ Wallet ทำกำไรจาก Memecoin

ข้อสังเกต: Memecoin ไม่ใช่การลงทุน แต่เป็น "การพนัน" ถ้าจะเล่น ให้ใช้เงินที่พร้อมเสียทั้งหมด ไม่เกิน 1-2% ของพอร์ต

31.8 ICO/IDO/Airdrop Scam

ICO Scam แบบคลาสสิก:

Step 1: สร้าง Whitepaper สวย คัดลอกหรือซื้อ Template
Step 2: จ้าง "ที่ปรึกษา" ปลอม (รูปจาก Stock Photo)
Step 3: เปิดระดมทุน ICO รับ ETH/USDT
Step 4: ระดมได้ $10M-$100M
Step 5: หาย — ไม่ส่งมอบโปรเจกต์ ไม่คืนเงิน

Airdrop Scam:
- เหยื่อได้รับ Token "ฟรี" ในกระเป๋า
- เหยื่อพยายามขาย Token นั้น ต้อง Connect Wallet กับ Smart Contract
- Smart Contract ดูดเงินทั้งหมดจากกระเป๋า

Phishing Airdrop:
- ส่ง Token ที่มีลิงก์ "เคลม" ใน Metadata
- เหยื่อกดลิงก์ → ไปหน้าปลอมของ Uniswap/PancakeSwap
- เหยื่อ Connect Wallet + Approve → กระเป๋าถูกดูด

คำเตือน: Token ที่ปรากฏในกระเป๋าของคุณโดยไม่ได้ขอ — ห้ามแตะต้องเด็ดขาด อย่าพยายามขาย อย่ากดลิงก์ใน Metadata

31.9 Influencer Pump

กลไก:
1. ผู้ปั่นจ่ายเงินให้ Influencer ดัง (Twitter/X, YouTube, TikTok)
2. Influencer "แนะนำ" Token โดยไม่เปิดเผยว่ารับเงิน
3. Followers แห่ซื้อ ราคาพุ่ง
4. Influencer และผู้ปั่นเทขาย
5. ราคาดิ่ง

สัญญาณเตือน Influencer Pump:
- Influencer พูดถึง Token แบบ "ฉับพลัน" โดยไม่เคยพูดมาก่อน
- ใช้คำศัพท์เหมือนกันหมดในหลายๆ Influencer (Script เดียวกัน)
- พูดถึง "100x potential", "Next Bitcoin", "Don't miss out"
- ไม่พูดถึงความเสี่ยงเลย
- พูดถึง Token ที่ Market Cap < $50M

กรณีดัง:
- 2018: DJ Khaled และ Floyd Mayweather ถูก SEC ปรับฐานโฆษณา ICO โดยไม่เปิดเผยว่ารับเงิน
- 2022: Kim Kardashian ถูก SEC ปรับ $1.26 ล้าน ฐานโพสต์โปรโมท EthereumMax โดยไม่เปิดเผย

31.10 MEV (Maximal Extractable Value) Manipulation

MEV คืออะไร: กำไรที่ Validator/Miner สามารถดึงออกมาได้จากการจัดเรียง Transaction ใน Block

รูปแบบ MEV:

ประเภท กลไก ผลกระทบต่อรายย่อย
Sandwich Attack ใส่ Tx หน้า-หลัง ซื้อแพง ขายถูก
Arbitrage จับส่วนต่างราคาข้าม DEX ทำให้ราคาเท่ากัน (กลาง)
Liquidation MEV จับ Liquidation ก่อน เหยื่อถูก Liquidate ใน Bad Price
Time-Bandit Attack Re-org Block เพื่อแย่ง MEV ทำให้ Tx Confirm ช้า
JIT Liquidity Add+Remove Liquidity ในบล็อกเดียว กิน Fee ของ LP จริง

Total MEV Extracted (จาก Flashbots):
- 2021-2024: รวมกว่า $1.6 พันล้าน ถูกดึงออกจากผู้ใช้ DeFi
- ส่วนใหญ่เป็น Sandwich Attack (60%)

31.11 Stablecoin Depegging Manipulation

รูปแบบ: ผู้ปั่นโจมตี Stablecoin ให้หลุดราคา $1 เพื่อทำกำไรจากการ Short

กรณีดัง: UST/Luna Crash (พฤษภาคม 2022)

ก่อนวิกฤต:
├── UST = Algorithmic Stablecoin มี Market Cap $18B
├── ผูกราคากับ LUNA ผ่าน Mint/Burn Mechanism
└── Anchor Protocol จ่าย Yield 20% ดึงดูดเงินมหาศาล

5 พฤษภาคม 2022:
├── มีการเทขาย UST $200M ใน Curve Pool
├── UST Depeg เล็กน้อย ($0.985)
└── คนเริ่มกลัว เริ่ม Withdraw จาก Anchor

7-9 พฤษภาคม 2022:
├── ความกลัวขยายตัว (Bank Run)
├── คน Mass Withdraw $5B ออกจาก Anchor
├── UST Depeg หนักขึ้น ($0.60)
└── Mint LUNA เพื่อ Burn UST → LUNA Hyperinflation

10-15 พฤษภาคม 2022:
├── LUNA จาก $80 → $0.0001 (-99.99%)
├── UST จาก $1 → $0.10
├── Market Cap หายไป $40 พันล้าน
└── นักลงทุนหลายคนสูญเสียทั้งชีวิต

บทเรียน:
1. Algorithmic Stablecoin ไม่ใช่ Stable
2. Yield สูงผิดปกติ (20%) ไม่มีในโลกความเป็นจริง
3. การพึ่งพา Reflexivity (ราคา A กระทบ B กระทบ A) คือบ้านไพ่
4. แม้ Tier 1 Project ก็ล่มได้

31.12 Smart Contract Exploit + Manipulation

รูปแบบ: Hacker ค้นพบช่องโหว่ของ Smart Contract แล้วใช้ประโยชน์

ตัวอย่างที่เคยเกิด:

ปี โปรเจกต์ มูลค่าเสียหาย รูปแบบ
2016 The DAO $60M Reentrancy Attack
2021 Poly Network $611M Cross-chain Bridge Bug
2022 Ronin Bridge $625M Validator Key Compromise
2022 Wormhole $326M Signature Verification Bug
2022 BNB Bridge $586M Proof Verification Bug
2023 Euler Finance $197M Donation Attack
2023 Multichain $126M Private Key Leaked

บทเรียน:
- แม้โปรเจกต์ใหญ่ที่ Audited ก็ถูก Hack ได้
- Cross-chain Bridge เป็นจุดเสี่ยงที่สุด
- Smart Contract ไม่มี "Customer Service" ให้กดอุทธรณ์

บทที่ 34: Position Sizing สำหรับ Crypto

ความผันผวนสูง = ขนาดต้องเล็กลง

Asset Class       Position Sizing Guideline
───────────────────────────────────────────
หุ้น Blue-chip    5-15% ของพอร์ต
หุ้น Mid-cap      3-8%
หุ้นเล็ก/IPO      1-3%
BTC/ETH          5-15% (ของพอร์ต Crypto)
Top 20 Altcoins  2-5%
Mid-cap Alt      1-3%
Small-cap Alt    0.5-1%
Memecoin/IDO     0.1-0.5% (เงินที่พร้อมเสีย)

กฎความเสี่ยงสำหรับ Crypto Portfolio

1. Crypto ไม่ควรเกิน 5-15% ของ Net Worth ทั้งหมด
   (สำหรับคนทั่วไป — ถ้ารับความเสี่ยงสูงได้ อาจถึง 25%)

2. ภายใน Crypto Portfolio:
   ├── 60-70% ใน BTC + ETH
   ├── 20-30% ใน Top 20 Altcoins  
   ├── 5-10% ใน High-conviction Plays
   └── 0-5% ใน Speculative (Memecoin, New Launch)

3. ห้ามใช้ Leverage > 3x — แม้แต่ใน BTC
   (Leverage 10x + ราคาขยับ 10% = Liquidation)

4. Stop Loss ต้องตั้งไกลกว่าหุ้น (15-25%) เพราะ Volatility สูง

5. ห้ามถัวเฉลี่ยขาลงในเหรียญที่ลง > 50%

บทที่ 35: กฎ 15 ข้อสำหรับนักลงทุน Crypto

1. ไม่ลงทุนในสิ่งที่อธิบายให้คนอื่นไม่ได้
2. Custody is King — Not Your Keys, Not Your Coins
3. ไม่เก็บ Crypto บน Exchange เกิน 6 เดือน
4. ใช้ Hardware Wallet สำหรับยอด > $1,000
5. ห้ามเชื่อ Influencer 100% — ทุกคนมี Bag
6. กระจายความเสี่ยง — ไม่ทุ่ม All-in กับ Token เดียว
7. รู้จัก Security Risk: Phishing, Scam, Hack
8. ไม่ Approve Smart Contract ที่ไม่รู้จัก
9. ตั้ง 2FA ทุกบัญชี (ใช้ Authenticator App ไม่ใช่ SMS)
10. ห้าม Click Link จาก DM
11. ห้าม Share Seed Phrase กับใคร — แม้แต่ Customer Service
12. ระวัง Romance Scam (Pig Butchering)
13. มี Exit Strategy ก่อนเข้า — รู้ว่าจะขายเมื่อไหร่
14. Take Profit เป็นระยะ — ไม่รอ Top
15. พักจาก Crypto บ้าง — มันไม่ได้สำคัญที่สุดในชีวิต

บทส่งท้าย: ทำไมคนส่วนใหญ่ขาดทุนใน Crypto

จากการศึกษาของ Bank of International Settlements (BIS) ปี 2022:

73% ของ Retail Investor ขาดทุนใน Crypto ตั้งแต่ปี 2015-2022

เหตุผล 5 ข้อหลัก:
1. เข้าตอนกระแสร้อน — ซื้อใกล้จุดสูงสุด ขายใกล้จุดต่ำสุด
2. ไม่เข้าใจสิ่งที่ลงทุน — ซื้อตามที่คนอื่นบอก
3. ใช้ Leverage มากเกินไป — Liquidation บ่อย
4. ตกเป็นเหยื่อ Scam — Rug Pull, Phishing, Hack
5. ไม่มีระบบการลงทุน — ตัดสินใจตามอารมณ์

คำพูดสุดท้าย: Crypto คือเทคโนโลยีที่ทรงพลังและเปลี่ยนโลกได้ — แต่ตลาด Crypto คือสนามที่ไม่ปราณี ใครก็ตามที่เข้ามาโดยไม่มีความรู้และวินัย จะออกไปด้วยกระเป๋าที่เบากว่าตอนเข้ามา หนังสือเล่มนี้ไม่ได้บอกให้คุณห่าง Crypto — มันบอกให้คุณ เคารพ Crypto

"ในตลาดหุ้น คุณต่อสู้กับเจ้ามือ ในตลาด Crypto คุณต่อสู้กับเจ้ามือ Bot Hacker และตัวเอง — พร้อมกัน"
